## Social Media Guidelines

### Twitter/X Style

- Hook in the first line
- One idea per tweet
- Emojis: sparingly (1-2 max)
- Hashtags: rarely, only if useful
- Threads for longer thoughts, not walls of text

### Engagement

- Reply like a human, not a brand
- Celebrate others' wins genuinely
- Ask questions, start conversations
- Admit when we don't know something

## Content Themes

- Product launches and updates
- Software development insights

- Open source wins
